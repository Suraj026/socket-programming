# Network Programming 

Two network programming projects built from scratch using raw sockets — no high-level networking libraries or system utilities under the hood.

| Part | Description | Language |
|---|---|---|
| [Part 1 — UDP Video Streaming](udp_video_server/README.md) | DNS resolution over TCP + live webcam streaming over UDP | Python |
| [Part 2 — NetProbe Toolkit](netprobe/README.md) | `ping`, `iperf`, and `traceroute` reimplemented using raw sockets | C |

See each part's README for full protocol design, usage, and implementation details.
